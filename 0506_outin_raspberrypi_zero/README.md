## Overview
OUT/IN Shield for Raspberry Pi Zeroは、Raspberry Pi Zeroと各種センサーやボタンをケーブルを1本接続するだけで使えるRaspberry Pi Zero対応シールドです。

Raspberry Pi Zero本体は含まれません。別途、お買い求めください。

- A3
- A4
- A5
- A6
- A7
