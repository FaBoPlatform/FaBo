# モーターギヤボックス前準備

ギヤ比は 114.7 : 1　組み立てます。

![](./img/motor_gear.png)

六角シャフトは、SINGLE GEARBOX付属のシャフトから、150mmシャフトに取り替えます。

![](./img/replace01.png)

モーターも、SINGLE GEARBOX付属のモーターから、6Vモーターに取り替えます。

![](./img/replace02.png)

完成図

![](./img/motor.png)

<hr>

# タイヤ前準備

前タイヤと後タイヤを各２本、計４本準備します。

前タイヤはゴム枠だけ組み立てます。

後タイヤはシャフト取り付け治具まで組み立てます。ホイール内部が深いほうが内側になります。

![](./img/tire01.png)

![](./img/tire02.png)

# サーボの組み立て


![](./img/servo01.png)

![](./img/servo02.png)

ギヤを中心（初期位置）にします。

方法

![](./img/type1_02_1.png)

クランクパーツを取り付け、手順１～３のように動かし、中心位置にします。

![](./img/type1_02.png)

手順４でネジで固定します。

向きに注意してください。

![](./img/servo03.png)

使用ネジ（M3-12ｍｍ）1本

# マイコンをマウントするネジの取り付け

![](./img/type1_04.png)

取り付けるマイコンによってネジ穴が違います。ガイドに合わせて取り付けてください。

裏から差し込んで固定します。

使用ネジ（M2.6-15ｍｍ）４本

<hr>

# サーボの取り付け

![](./img/type1_05.png)

上から固定パーツで挟むように取り付けます。

サーボの向きに注意してください。

使用ネジ（M3-12ｍｍ）２本

<hr>

# ステアリングパーツの取り付け

![](./img/type1_06.png)

タイヤを取り付けるパーツを組み立てます。

裏からM3ナットを差し込みます。

（構成部品として完成している場合があります）

2個準備します。

![](./img/type1_07.png)

ステアリングバーを取り付けます。

サーボのクランプパーツの溝にはめるように取り付けます。

![](./img/type1_08.png)

フロントパーツで挟むように固定します。

使用ネジ（M3-25ｍｍ）4本

![](./img/type1_09.png)

![](./img/type1_10.png)

<hr>

# バッテリーキャリアの組み立て

![](./img/type1_11.png)

バッテリー落下防止用ネジを裏から２か所取り付けます。

使用ネジ（M3-12ｍｍ）

![](./img/type1_12.png)

使用ネジ（M3-35ｍｍ）4本

# ギヤボックスの取り付け

上部シャーシの後部裏に取り付けます。

使用ネジ（M3-12ｍｍ）２本

![](./img/type1_13.png)

<hr>

# 前タイヤの組み立て

２本組み立てます。

![](./img/type1_14.png)

内部にベアリングを２個をいれて、アクリルスペーサーで挟むように固定します。

ホイールが深い方が内側になります。

使用ネジ（M3-35ｍｍ）各１本

<hr>
