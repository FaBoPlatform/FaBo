![](./img/circlebeacon_00.jpg)

## 組み立て方法
1.裏返し状態で、(1)、(2)、(3)の順番で重ね、ネジを(1)から通す。

![](./img/circlebeacon_01.jpg)

方向に注意してください！

![](./img/circlebeacon_02.jpg)

2.ビーコンを裏返し状態でセットする。

![](./img/circlebeacon_03.jpg)

※(3)の枠にハメ込む

3.(4)、(5)、(6)の順番で重ね、最後に(7)をセットする。

![](./img/circlebeacon_04.jpg)

4.ナットで固定して完成です。
