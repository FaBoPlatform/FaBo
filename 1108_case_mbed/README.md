## パーツ構成

![](./img/mbed_00.jpg)

 
(1)が一番下、(6)が一番上になります。

使用ネジはM3-25プラネジ４本です。

## 組み立て方法
1.　下から(1)、(2)、(3)、(4)を順番に重ねます。

![](./img/mbed_01.jpg)

組み立て易くする為に、下から四方の穴にネジを差し込んでおきます。

ナットでの固定はしません。

2.　(4)の内枠に合わせるようにボードをセットします。

![](./img/mbed_02.jpg)

ボードを(4)にハメ込む形になります。

3.　(5)、(6)を順番に重ねます。

(5)がボードを抑える形になります。

4.　ネジをナットで固定します。ネジの方向は使用環境に応じて変えてください。

ゴムクッションはお好みでお使いください。

ネジが長いと感じる方はニッパーなどでカットしてください。
